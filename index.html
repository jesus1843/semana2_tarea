Descripción

Este es un diseño básico de una página web de una tienda de muebles

Tiene combinación del framework de CSS Bootstrap en su versión 4.6, más unos estilos CSS personalizados escritos por mí

Los estilos CSS escritos por mi fueron para usar CSS Grid a la lista de productos, para agregarle un espaciado entre cada producto, ya que esa opción no está disponible con las clases CSS de Bootstrap
